# CodeRenga 名称とコンセプト

## 名称の由来

CodeRenga は、日本の伝統的な連作詩「連歌（Renga）」に由来する。

連歌は一人で完結させるものではない。複数の詠み手が前の句を受け取り、その文脈をつなぎながら次の句を重ねていく表現形式である。

CodeRenga も同じように、クラウド LLM が設計や方針を考え、ローカル LLM がその意図を受け取って実装し、ツールが差分・実行・検証をつなぐ。

CodeRenga は、ひとつの AI にすべてを任せるのではなく、複数の知性と実行環境を連携させながら、コードを一節ずつ形にしていくためのローカル開発エージェントである。

## タグライン

**Cloud LLM thinks. Local LLM builds. CodeRenga links the verses.**
